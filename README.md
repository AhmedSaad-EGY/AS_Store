# 🛒 AS Store - Modern E-Commerce SPA

A sleek, responsive e-commerce Single Page Application (SPA) built with **Vue.js 3** and **Bootstrap 5**.

## 🚀 Features
* **SPA Architecture**: Smooth navigation between Home and Cart without page refreshes.
* **Live Search**: Instant product filtering by name.
* **Reactive Cart**: Add/remove items with automatic total price calculation.
* **Responsive Design**: Fully optimized for mobile and desktop views.

## 🛠 Tech Stack
* **Vue.js 3** (CDN)
* **Bootstrap 5.3**
* **Bootstrap Icons**
